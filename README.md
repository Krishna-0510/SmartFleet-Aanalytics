# SmartFleet-Aanalytics
🚚 SmartFleet Analytics is a 📊 R Shiny–Powered Fleet Dashboard analyzing 5,000+ delivery records with Real-Time KPIs, 🚦 Delay Prediction, 🗺️ Route Optimization, 👨‍✈️ Driver Analytics, and 💰 Cost Insights using Shiny, Plotly, Highcharter, DT, and Tidyverse to solve delivery delays and fleet inefficiency.
